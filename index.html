Eye-Controlled Games

HCI Project

For our project, we are implementing games that can be played with only the use of your eyes as input. We are using a package called WebGazer.js. We are starting with implementing a version of Agar.io, and if time permits, we plan on possibly implementing another game as well.

For the best gameplay experience, start with the calibration so the eye tracking software can optimize to your eye movement.
