Eye-Controlled Games

HCI Project

For our project, we are implementing games that can be played with only the use of your eyes as input. We are starting by implementing a single-player version of Agar.io that can be played by downloading the eye tracking software here. Our plan is to use this to test out the game using eye-tracking as input, but switch to an eye-tracking software that we can integrate into the browser.
